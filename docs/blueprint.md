# **App Name**: LensVivid

## Core Features:

- Video Header: Display an introductory video at the top of the landing page.
- Image Gallery: Showcase photography portfolio with a visually appealing gallery layout.
- Scrolling Text & Images: Implement a scrolling section with text and images about the photography services offered.
- Firebase Authentication: Enable user sign-in with Google using Firebase.
- About Section: Introduce the photographer/team and their approach to photography.

## Style Guidelines:

- Primary color: Deep purple (#673AB7) to evoke creativity and professionalism.
- Background color: Very dark grey (#1E1E1E) to provide a modern and sophisticated backdrop.
- Accent color: Electric blue (#7DF9FF) for highlights and interactive elements.
- Headline font: 'Poppins', sans-serif, for a contemporary, geometric title.
- Body font: 'PT Sans', sans-serif, for easily readable blocks of text.
- Employ a layout where a header video introduces the site and below that is a gallery of images to display Photography
- Utilize a "glass vibe" using a semi-transparent background overlay to add depth.
